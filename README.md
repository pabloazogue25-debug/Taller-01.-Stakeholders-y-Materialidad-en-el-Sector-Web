#  Análisis de Materialidad ASG en Desarrollo Web

##  Objetivo
Este proyecto analiza los **grupos de interés** y los **temas materiales ASG** en una consultora de desarrollo web, aplicando criterios de sostenibilidad.

## 👥 Stakeholders Identificados
- **Clientes empresariales**
- **Usuarios finales**
- **Equipo de desarrollo**

##  Temas Materiales
- *Eficiencia energética del software* (Ambiental)
- *Privacidad y protección de datos* (Gobernanza)
- *Accesibilidad digital* (Social)

##  Tecnologías utilizadas
- **XML** para estructuración de datos
- **GitHub** para control de versiones
- **VS Code** como entorno de desarrollo
